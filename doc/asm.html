CREATING AN ASSEMBLER WITH TELOCIDESI

THE GLOBAL ASSEMBLER ARCHITECTURE

PROCESSOR SPECIFICATIONS :
  • 3 Registries : REG- , REG0 , REG+
  • RAM : 3-trits adresses (0 to 26)
  • Wordsize : 9 trits
  • Sign Flag S : sign of REG0

BASIC ASSEMBLY INSTRUCTION SET

(% : Registry // @ : RAM)
Name Opcode Operands Role Description
Halt HLT pass Move to next line.
Direct Load DLD %, val val → REG Load the value in the registry.
Move MOV %1, %2 REG → REG Move the first registry in the second one.
Load LOD %, @ RAM → REG Load a value from the RAM to the Registry.
Store STR @, % REG → RAM Store the value from the Registry to the RAM.
Sum ADD REG0 = REG++REG- Load in REG0 the result.
Substraction SUB REG0 = REG+-REG- Load in REG0 the result.
Jump JMP @ Inconditionnal Jump to the @ adress.
Jump If JIF @1, @2 JMP @1 if S = + // JMP @2 if S = -
